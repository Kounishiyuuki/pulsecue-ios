## Key Features

### Interaction Patterns

1. **Swipe Actions**
   - Swipe left on routine: Pin/Unpin, Duplicate
   - Swipe right on routine: Delete
   - Swipe on step: Delete

2. **Gestures**
   - Long press routine: Context menu (Start, Pin, Duplicate, Delete)
   - Drag and drop: Reorder steps in routine editor

3. **Haptic Feedback**
   - Impact feedback when starting/stopping
   - Success notification on completion
   - Light impact for +10s and Back actions
   - Selection feedback for toggles

4. **Audio Cues**
   - Beep sound on step transitions (toggleable)
   - System sound for notifications

5. **Notifications**
   - Local notifications scheduled for step completion
   - Delivered even when app is in background

### State Persistence

- **Runner State** saved to UserDefaults:
  - Current routine ID
  - Current step index
  - Deadline date (for timer recovery)
  - Running/resting state
  - Elapsed seconds

- **SwiftData** for persistent storage:
  - Routines and Steps
  - DayLog entries

### Color Scheme

- **Primary Actions**: Blue/Accent Color
- **Positive Actions**: Green (Complete, positive balance)
- **Warning Actions**: Orange (Skip)
- **Destructive Actions**: Red (Stop, Delete)
- **Secondary Actions**: Gray (Back, disabled)
- **Highlights**: Accent color with opacity for active state

## Technical Notes

- iOS 17+ required for SwiftData support
- No external dependencies
- Fully offline-capable
- MVVM architecture
- Service-oriented design for modularity
- Type-safe SwiftData models with relationships
